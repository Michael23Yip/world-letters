# World Letters

World Letters is a public platform where individuals can write open letters to the world, or to people in specific countries or regions, and receive replies.

## Getting Started

1. Clone the repository.
2. Install dependencies.
3. Run the application.
## Supabase Setup

1. Create a Supabase project.
2. In the SQL Editor, run the statements in `supabase/schema.sql`.
3. Run the statements in `supabase/policies.sql`.
4. In Authentication settings, enable Email (Magic Link).
5. Use the Publishable key as the value for `SUPABASE_ANON_KEY` later.
